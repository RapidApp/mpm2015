mpm2015
=======

"Fun with RapidApp" presentation for Brew City Perl Mongers (Milwaukee.pm),
April 16, 2015

## usage

```bash
# Clone the repo:
git clone https://github.com/RapidApp/mpm2015
cd mpm2015/
git submodule update --init

# Install required packages:
cpanm --installdeps .

# Start-up app.psgi:
plackup

```
